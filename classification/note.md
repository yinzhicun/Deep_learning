# <center>Classification(分类)</center>

## 简介
- 预设定相应的**函数模型**
- 通过已有的数据求解函数的参数，得到函数的关系式
- 对未知的数据进行预测
- 与Regression不同的是，Classification会对函数的输出进行离散化处理，以满足其特殊的要求
