# 七篇玻璃去反光论文的总结(顺序按阅读顺序排序)

## 1. CEILNet

本文通过强调图片中的边缘信息，通过两个网络级联解决了图片去反光问题和图片平滑问题，**创新点在网络结构方面**。从中我关注到了这些：

- 传统算法处理去反光问题的方式
- 图像的三个评价标准MSE、PSNR、SSIM
- 带反光图像的数学模型，通过数学模型生成训练集

## 2. Single Image Reflection Separation with Perceptual Losses

本文通过引入了基于特征的损失函数，基于与真实去反光图片对抗模型的损失函数，和基于反光层与去反光层边界互斥的损失函数解决了图片去反光问题，**创新点在新型损失函数的提出**。从中我关注到了这些：

- 基于VGG的hypercolumn features
- 空洞卷积
- GAN网络生成损失函数
- 基于边缘冲突假设的像素级别的损失函数，其中的下采样操作和tanh函数的巧妙使用
